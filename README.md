# touchdesigner-visualizations

## recurse

TouchDesigner project. Edited live via a local MCP bridge (`.mcp.json` -> `touchdesigner`
server at `http://localhost:40404/mcp`) rather than by hand-editing the `.toe` binary.

The actual `recurse.toe` file is created/saved from within TouchDesigner (or via the MCP
bridge's project tools) and lives in `recurse/`.
